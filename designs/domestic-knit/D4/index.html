PATTERN D4

  • Size: 100 × 100 px
  • Color: 2

INSTRUCTIONS AND SETUP

A/ YARN
  • Color A: 3x cotton tourquise 30/2
  • Color B: 1x elastan grey 28/2
B/ SIZE
  • Needles: 100
  • Rows: 110
C/ CARRIAGE SETUP

START:

First Row

  • Main bed: Tension: 0.1; Buttons: All out; Mode: N
  • Ribber: Tension: 0.1; Cam levers: All down; Lilli buttons: I; Lower slide lever: II

Second and Third Row

  • Main bed: Tension: 0.2; Buttons: Right part in; Mode: N
  • Ribber: Tension: 0.2; Cam levers: Left part up; Lilli buttons: I; Lower slide lever: I

AYAB KNITTING:

  • Main bed: Tension: 1.2; Buttons: Both part in; Mode: KC2
  • Ribber: Tension: 1.2; Cam levers: Both part up; Lilli buttons: lilli; Lower slide lever: lilli

Swatch D4A: Front

Swatch D4B: Back
